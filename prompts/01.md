Crie o componente Header da landing page seguindo EXATAMENTE o design da imagem anexada.

## Requisitos Visuais:
- Logo "KL PSICÓLOGA" à esquerda (usar /public/images/logo.png)
- Fundo bege claro (#F5F1ED ou similar do tema)
- Navegação à direita com links: "Início", "Sobre", "Serviços", "Contato"
- Typography: fonte serif elegante para links
- Espaçamento horizontal amplo entre os links
- Header fixo no topo (sticky)
- Sombra sutil embaixo do header

## Especificações Técnicas:
- Criar em: @app/_components/header.tsx (substituir o existente)
- Usar APENAS cores do tema de @app/globals.css
- Usar componentes shadcn/ui disponíveis em @app/_components/ui/
- Navegação com smooth scroll para as seções da página
- Mobile: hamburger menu com Sheet do shadcn
- Desktop: links inline

## Comportamento:
- Ao clicar nos links, fazer scroll suave até a seção correspondente
- Highlight no link da seção atual (usar IntersectionObserver)
- Responsivo: mobile (<768px) mostra menu hamburger

## Estrutura:
- Componente Server Component (sem 'use client' no início)
- Apenas o menu mobile precisa de interatividade (separar em componente)
